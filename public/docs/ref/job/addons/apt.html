Apt

APT packages and sources to install

Install APT packages and sources without using apt-get commands in a before_install script manually.

This node uses the key packages as a default prefix if not given a map.

See also

  • Installing Packages with the APT Addon

Types

  • Map
  • Sequence of String
  • String
  • Boolean

Keys

If given a map, the following keys are supported:

  • packages — Package names (type: sequence of string, or string, alias: package, e.g.: cmake)

  • sources — Package sources (type: sequence of map, sequence of string, map, or string, alias: source, e.g.: ubuntu-toolchain-r-test)

  • dist — Distribution (type: string)

  • update — Whether to run apt-get update (type: boolean)

  • config — (type: map)

Examples

apt:
  packages:
  - cmake
  sources:
  - ubuntu-toolchain-r-test
  dist: string

apt:
- cmake

apt: cmake


apt: true
