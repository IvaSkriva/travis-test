Code Climate

CodeClimate settings

This node uses the key repo_token as a default prefix if not given a map.

See also

  • Using Code Climate with Travis CI

Types

  • Map
  • Secure
  • Boolean

Keys

If given a map, the following keys are supported:

  • repo_token — Code Climate repo token (type: secure)

Examples

code_climate:
  repo_token:
    secure: encrypted string

code_climate:
  secure: encrypted string

code_climate: true
