Sauce Connect

SourceConnect settings

See also

  • Using Sauce Labs with Travis CI

Types

  • Map
  • Boolean

Keys

If given a map, the following keys are supported:

  • enabled — (type: boolean)

  • username — (type: secure)

  • access_key — (type: secure)

  • direct_domains — (type: string)

  • tunnel_domains — (type: string)

  • no_ssl_bump_domains — (type: string)

Examples

sauce_connect:
  enabled: true
  username:
    secure: encrypted string
  access_key:
    secure: encrypted string

sauce_connect: true
